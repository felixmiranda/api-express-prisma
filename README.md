# API Express-Prisma

Demo de un api usando express, prisma orm

## Instalación

Utilice el gestor de paquetes [npm](https://docs.npmjs.com/downloading-and-installing-node-js-and-npm) para instalar los paquetes.

```bash
npm install
npm run generate-schema
```

## Ejecución

```bash
npm run postinstall
npm run start
```
